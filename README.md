# arch
It's My Installer Scripts Of Arch For Dwm....

Note :=  This Installer Scripts Doesn't Include Nvidia Or AMD Graphics Drivers....
  1.)  If You Have Nvidia Graphics Card Then Add The "nvidia" Package Name Where The Base Package Name Are Written....
  2.)  If You Have AMD Graphics Card Then Replace The "intel-ucode xf86-video-intel vulkan-intel" With "amd-ucode xf86-video-amdgpu vulkan-radeon" Package Name Where The Base Package Names Are Written....
  
This Shell Script's Codes Are Copied From Bugswriter/arch-linux-magic/arch-linux-magic.sh....

But, I Modified His Scripts for UEFI / BIOS / Virtual Machines Installations..&..Added Some More Options Like Creating SWAP Partitions & Customised DWM Installation Also....
